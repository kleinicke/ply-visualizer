3D Point Cloud & Mesh Visualizer

View and inspect 3D files directly in VS Code with interactive visualization, camera controls, and measurements.

  • Supports PLY, OBJ, STL, PCD, PTS, OFF, GLTF/GLB formats
  • Depth image to point cloud conversion (TIFF, PNG, NPY, PFM)
  • Camera parameter support (intrinsics, calibration files)
  • Distance measurement tools and rotation center controls
  • Multiple camera conventions (OpenCV, OpenGL)
  • Web version available — try it without installing

Float & TIFF Visualizer

Analyze float and integer TIFF images with advanced visualization controls — also supports EXR, NPY, PNG, JPG, PPM, PFM and PGM.

  • Multi-channel and floating-point TIFF support
  • Interactive pixel inspection (hover for exact values)
  • Dynamic normalization range adjustment
  • Gamma and brightness correction controls
  • Export visualization as PNG
  • Supports compressed TIFF (Deflate, LZW with predictors)
